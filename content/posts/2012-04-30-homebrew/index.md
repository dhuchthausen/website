Good news for all mac users. SCM-Manager is now available via <a title="Homebrew" href="http://mxcl.github.com/homebrew/" target="_blank">homebrew</a>.

Just install SCM-Manager with:

<pre>  brew install scm-manager</pre>

And start it with:

<pre>  scm-server</pre>
